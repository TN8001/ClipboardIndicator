# ClipboardIndicator
![アプリスクリーンショット](https://github.com/TN8001/ClipboardIndicator/AppImage.png)
## 概要
クリップボードに文字が入っているかを確認する常駐型アプリ。
## 特徴
* 最低限コピー出来たかを確認するだけの小ささ（大きくも出来ますが）
* 小さいので起動時に見失わないためのアピール機能（解ってしまえば大したことはしていませんが 実装に苦労したので参考にして頂けたら幸いです）
* クリップボードのクリア機能
## 使い方
1. 起動する
1. 好きな位置に移動する
1. 右クリックで設定を好きに変える
1. お好みでスタートアップに入れてください
